# Custom Vite Template

React 학습 시작을 위한 사용자 정의 템플릿

```sh
degit (경로)/likelion-vite-react <폴더_이름>
cd <폴더_이름>
pnpm install
```

## 실행 명령어

개발 서버 구동

```sh
pnpm dev
```

프로젝트 빌드

```sh
pnpm build
```

프로젝트 프리뷰

```sh
pnpm preview
```

린팅 검사

```sh
pnpm lint
```

포멧팅

```sh
pnpm format
```

## Git 관련 명령어

Git 훅(hook), pre-commit 훅을 건너뛰도록 설정

```sh
git commit -m "Your commit message" --no-verify
```
